Supported Operating Systems

connect.com.au supports a wide variety of common operating systems.

Disks with these software packages can be provisioned with an API call containing one of the following possible values:

  • ubuntu1204 for Ubuntu 12.04
  • centos6 for CentOS 6
  • win2008 for Windows Server 2008
  • win2012 for Windows Server 2012
